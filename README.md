# SentinelEdge Endpoint Security – Documentation

This repository contains end-to-end product documentation for **SentinelEdge Endpoint Security**, a fictional but realistic endpoint protection platform.

## Project Purpose
This project demonstrates:
- Security product documentation best practices
- Docs-as-code using Markdown
- Documentation planning and tracking using GitHub Projects
- Microsoft Style Guide and DDLC alignment

## Product Overview
SentinelEdge Endpoint Security protects endpoints against:
- Exploit-based attacks (e.g., EternalBlue-style exploits)
- Ransomware and malware
- Unauthorized file and process activity

## Documentation Scope
The documentation includes:
- Product and architecture overview
- Installation and setup guides
- Feature documentation (Exploit Protection, Ransomware Protection)
- Alerts, logs, and troubleshooting
- Security-focused release notes

## How This Repo Is Organized
- `/docs` – Markdown documentation files
- GitHub Projects – Documentation planning and lifecycle tracking

> Note: SentinelEdge is a fictional product created solely for portfolio and learning purposes.
